HoneyNet

Mesh-networking IoT. Sweet like honey.

HoneyNet is a simple, intuitive toolkit on which to build your next IoT system. Its sturdy, lightweight framework takes care of radio communication so you can focus on sensor-node development.

A Sensor Node in 5 minutes

#include "HoneyNode.h"
#define ANALOG_CH 1
HoneyNode node(7, 8);

void setup() {
    node.begin();
    node.AddChannel(ANALOG_CH, uint16_t);
}

void loop() {
    uint16_t value = AnalogRead(A0);
    node.Send(ANALOG_CH, value);
    delay(1000);
}
